Passo a Passo, para subir documentos para o GitHub.

Após fazer todas alterações da pagina ou arquivos,

se utilizar o "git status" ira aparecer uma frase vermelha informando que houve alterações que não foram salvas.

Assim é necessario ir utilizar "git add . ou git add -a" (para adicionar todos os arquivos)

Apos isso caso queira saber se deu certo, usa o "git: status", que ira mostrar o status.

Depois disso, tem que utilizar o "git commit -m "colocar um comentario aqui dentro, ex: inclusão de anotações""

Para finalizar, é necessario dar um push, para subir os arquivos/commit para a nuvem.

para isso tem que usar o "git push origin main"
